Module name: Autotrading
Short name: autd
OS: Windows 7 +
Platform, framework: MQL 5, .NET(?)

Description: роль модуля в следующем - проверка конфигурации системы, загрузка необходимых данных с сервера для анализа возможности выставления ордеров, при открытии/закрытии ордеров сохранять на сервере информацию о совершенных сделках.
